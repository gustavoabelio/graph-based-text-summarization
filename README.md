# Graph-Based Automatic Text Summarization with TextRank

A graph-based extractive text summarization system built with Python using the TextRank algorithm.

The project demonstrates how Graph Theory can be applied to Natural Language Processing (NLP) by implementing the TextRank algorithm, an unsupervised graph-based ranking method for extractive text summarization.

TextRank models a document as a weighted graph in which sentences become vertices and semantic similarities become weighted edges. It then applies the PageRank algorithm to rank sentence importance, selecting the most relevant ones to compose the final summary. Unlike approaches based on LLMs or Generative AI, this method relies entirely on Graph Theory and statistical text analysis.

---

## Graph Modeling

Each input document is modeled as a weighted graph.

- **Vertices:** individual sentences.
- **Edges:** semantic similarity between sentence pairs.
- **Edge Weights:** Cosine Similarity calculated from TF-IDF vectors.

This representation allows the PageRank algorithm to identify the most central sentences in the graph, which are selected to compose the final summary.

## Algorithm

The implementation follows the classic TextRank workflow:

1. Sentence segmentation
2. Text preprocessing
3. TF-IDF vectorization
4. Cosine Similarity computation
5. Weighted graph construction
6. PageRank execution
7. Sentence ranking
8. Extractive summary generation

## Output

After execution, the application generates the following files inside the output directory:

- **resumo.txt** — Extractive summary generated by the TextRank algorithm.
- **grafo.png** — Visualization of the sentence graph.

The graph visualization represents:

- **Node size:** proportional to the PageRank score.
- **Edge thickness:** proportional to Cosine Similarity.
- **Central nodes:** the most relevant sentences identified by the algorithm.
